Задача про нарізання пирога із родзинками.
Умову (сам пиріг з родзинками) можна задати в самоу коді, або ж в консолі.
Для задання пирога використовуйте:
для позначення пустого місця в пирозі: .
для позначення місця в пирозі з родзинкою: o
Для розмежування строк(рівнів) в пирозі, використовуйте \n при задачі в коді програми, та пробіли при заданні пирога в командному рядку
Приклад:
Console: ruby CutTheCake.rb ........ .....o.. ..o..... ........
Code: cake = "........\n.....o..\n..o.....\n........"
Output:
........
.....o..
..o.....
........
P.S. Програма буде видавати результат із найбільшою можливою шириною першого елементу вихідного масиву шматків, навіть якщо інсує декілька способів розрізати пиріг на одакові за розміром шматки (щоб в кожному була одна родзинка).
P.P.S. Навіть якщо заданий пиріг відповідає вимогам (має прямокутну форму та кількість родзинок більше 1 та менше 10), іноді розділити його його на потрібні щматки буде неможливо.
